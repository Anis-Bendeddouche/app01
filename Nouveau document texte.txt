## 🎯 Vue d'ensemble du projet

### Objectif
Développer un outil (SaaS ou application) capable de transformer **n'importe quel site WordPress** en application mobile native (Android/iOS).

### Deux approches possibles

| Approche | Type | Complexité | Qualité | Acceptation Stores |
|----------|------|------------|---------|-------------------|
| **WebView** | Coque native qui charge le site | 🟢 Faible | 🟡 Moyenne | 🔴 Risquée (Apple) |
| **API Native** | App qui dessine nativement via API | 🔴 Élevée | 🟢 Excellente | 🟢 Garantie |

---

## 🌐 Approche 1 : WebView (Mode Simple)

### Concept
Créer une application mobile "vide" qui agit comme un navigateur web invisible. Elle charge l'URL du site WordPress fourni par l'utilisateur.

### Technologie recommandée : Capacitor (par Ionic)
Capacitor permet de créer une application web compilable en code natif (Android/iOS) avec la capacité de charger des URLs externes.
